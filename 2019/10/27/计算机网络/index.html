物理层

1.数字信道带宽计算: 奈奎斯特定理(无噪音)

奈奎斯特定理: $B=2W$ (B 为码元速率,波特率 W 为带宽)
信息量 n (位)与码元种类N: $n=log_2N$
数据速率:$R=logN=2WlogN$

2. 数字信道带宽计算:香侬定理(有噪音)

香侬定理:

$C = Wlog_2(1+S/N)$
其中,$C$为数据速率. $W$为信道带宽.$S$为信号的平均速率.$N$为噪声平均功率.$S/N$为信噪比

分贝数与信噪比之间的关系:

$dB=10log_{10}S/N$

3.一些数据

  • 单位换算:
    • 通信换算进率1000, 如1G = 1000M
    • 存储换算进率1024, 如1G = 1024M
  • 信道延迟:
    • 电缆信道延迟200m/us
    • 卫星信道延迟: 270ms

4. 编码方式

  • 归零性编码
    • 归零码

    • 双相码(电平翻转)

  • 曼码和差分曼码:[曼码降0升1(降1升0)效率50%；差分曼码折0平1，效率50%]

数据通信基础

1.通信方向

  • 单工:电视
  • 全双工:手机
  • 半双工:对讲机

2.交换方式

  1. 电路交换

    优点:独占性, 实时性, 适合传输大量的数据
    缺点:需要建立一条物理连接,利用率低

  2. 报文交换

    优点:不需要专门的通道, 线路利用率高, 存储转发节点可校验纠错
    错误:有通信时延

  3. 分组交换

    优点:利用率高,可选路径,数据率转换支持, 支持优先级
    缺点:时延,开销大

3.多路复用

  1. 频分复用FDM
    • 不同的频率，子信道隔离频带防串扰
  2. 时分复用TDM
    • 不同的时间，轮流占用
  3. 波分复用WDM
    • 不同的波长(光纤)

4.差错控制

1.检错码

奇偶校验是最常用的检错方法，其原理是在 位的 ASCII 代码后增加一位，使得码字中1的个数为奇数或偶数

广域网的概念和分类

广域网：广域网是指长距离跨地区的各种局域网、计算机、终端互联在一起，组成一个资源共享的通信网络。
